Delete Existing Data

You can delete data from tables here

Delete from Covid

Deleting an existing covid data requires the following fields formatted as follows:

  • Country: Country where the record was collected fromm. First letter of the country should be upper-case: United States
  • Date recorded: Date when the record was collected. Should be formatted as yyyy-mm-dd hh:mm:ss





Delete from Content

Deleting an existing content data requires the following fields formatted as follows:

  • ShowID: ID that can uniquely define a content. Should be formatted as 'sxxx', where a lower-case letter 's' is followed by positive integer greater than 1




Delete from Financial

Deleting an existing financial data requires the following fields formatted as follows:

  • filing_quarter_id: Date when the financial record was collected in one of the four quarters (03-31, 06-30, 09-30, 12-31). Should be formatted as yyyy-mm-dd hh:mm:ss
